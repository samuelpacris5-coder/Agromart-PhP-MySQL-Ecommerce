## 📌 Description
AgroMart is a full-stack agricultural eCommerce web application built with PHP and MySQL.  
It allows farmers and customers to buy and sell agricultural products through a secure and user-friendly online platform.

---

## 🚀 Features
- 👤 User registration & login system  
- 🛒 Shopping cart functionality  
- 🌾 Product listing & management  
- 📦 Order placement & tracking  
- 🔍 Product search and filtering  
- 📱 Fully responsive design  
- 🔒 Secure backend (prepared statements & validation)

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** Apache (XAMPP/WAMP)

## ⚙️ Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/agromart-php-mysql-ecommerce.git

2. Move to Server Directory

Place the folder inside:

C:\xampp\htdocs\

3. Start Server

Open XAMPP and start:

Apache ✅
MySQL ✅

4. Import Database
Open: http://localhost/phpmyadmin
Create database:
agromart_db
Click Import
Upload your .sql file

5. Configure Database Connection

Open db.php and update:

$host = "localhost";
$user = "root";
$password = "";
$database = "agromart_db";

6. Run the Project

Open browser:

http://localhost/agromart-php-mysql-ecommerce
